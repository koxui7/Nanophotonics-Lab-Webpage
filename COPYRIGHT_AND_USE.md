## What This Means

- The source is visible so collaborators can inspect it, discuss it, fork it on GitHub, and propose changes through pull requests.
- The absence of an open-source license means the repository contents are **not** granted for general reuse, redistribution, or unrestricted modification.

## Permitted Collaboration

The maintainer permits the following collaboration workflow:

- view the repository
- fork the repository on GitHub
- create branches in your own fork
- prepare changes and submit pull requests back to the upstream repository

## Restricted Uses

Unless the maintainer gives explicit permission, you may **not**:

- deploy this repository, or a modified version of it, as the public website of another lab, group, company, or organization
- redistribute modified copies as an alternative official version
- rebrand the repository as another group's website template
- present a modified version as an official Qiang Lin research group website without approval

## Intended Official Use

The official deployment and official maintained identity of this website are reserved for the Qiang Lin research group and its authorized maintainers.

## Review and Merge Policy

- Contributions are expected through pull requests into `main`.
- Final review and merge decisions for the upstream repository are made by the maintainer.

## Questions

If you want to use this repository outside the collaboration flow above, ask the maintainer for explicit permission first.
